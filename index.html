Surgimento da Internet

A história da internet teve início com a Arpanet, uma rede de troca de informações que havia sido desenvolvida para conectar instalações de pesquisas e militares com o Pentágono, nos Estados Unidos, na década de 1960. O crescimento dessa rede permitiu que seu uso se estendesse para fins privados e comerciais. A internet chegou ao Brasil somente no final da década de 1980.

A internet é atualmente o maior meio de comunicação utilizado planeta, com mais de 5 bilhões de usuários. Por ela, é possível enviar mensagens instantaneamente, movimentar dinheiro em contas bancárias, comprar mercadorias, acessar redes sociais, fazer pesquisas etc.

A internet chegou ao Brasil, em 1988, por meio de uma conexão entre um computador da Universidade Federal do Rio de Janeiro (UFRJ) e a rede da Universidade de Maryland, nos Estados Unidos, tornando-se pública e comercial entre 1994 e 1995. Atualmente, de acordo com a Pnad, cerca de 90% das residências brasileiras têm acesso à internet."

Como funciona o HTML

Quando navegamos na internet usando o computador ou dispositivos móveis, não nos damos conta de como todas as informações que vemos são traduzidas de códigos e comandos de programação.

No entanto, a maior parte dos conteúdos com os quais interagimos nas páginas da internet são concebidos usando HTML.

HTML, ou HiperText Markup Language, é o que permite criar a estrutura básica de um site na internet. Embora ele não seja propriamente uma linguagem de programação, usando marcações de códigos chamados “Tags”, ele possibilita que informações sejam organizadas e formatadas para dar estrutura e forma à página.

A estrutura de uma página criada em linguagem HTML é interpretada, editada e exibida pelos navegadores de internet, conforme os comandos das Tags foram configurados. Dessa forma, é graças à linguagem de HTML e aos navegadores que conseguimos acessar e interagir de forma apropriada com diversos conteúdos online.

O que são Tags

As tags HTML são divididas em várias categorias, cada uma com uma função específica. Aqui estão algumas das categorias mais comuns:

  • Tags de estruturação de texto:essas tags são usadas para estruturar o texto em parágrafos, títulos, listas e outros elementos. Exemplos: <p>, <h1> a <h6>, <ul>, <ol>, <li>, entre outras;

  • Tags de links e imagens: essas tags são usadas para criar links para outras páginas ou para incluir imagens. Exemplos: <a> para links e <img> para imagens

  • Tags de formulário: usadas para criar formulários interativos, como campos de entrada de texto, botões de opção e botões de envio. Exemplos: <form>, <input>, <select>, <textarea>, entre outras;

Próximos Passos

Após aprender HTML, muitos desenvolvedores iniciantes se perguntam qual deve ser o próximo passo na jornada de desenvolvimento web. Embora o HTML seja a espinha dorsal de qualquer página na internet, ele é apenas o ponto de partida para criar sites dinâmicos e interativos. O próximo passo natural é se aprofundar em CSS, a linguagem responsável por estilizar as páginas e torná-las visualmente atraentes. O CSS permite personalizar elementos da página, definir cores, tamanhos, margens e criar layouts. Com ele, é possível controlar a disposição dos itens na tela de maneira responsiva, garantindo que o site funcione bem em diferentes dispositivos, como smartphones, tablets e desktops. Ferramentas modernas como Flexbox e Grid Layout são essenciais para dominar a construção de layouts complexos de forma simples e eficiente.


Logo após a familiarização com CSS, surge a necessidade de aprender JavaScript, a linguagem que transforma páginas estáticas em interfaces dinâmicas e interativas. Com JavaScript, você pode criar funcionalidades que respondem às interações do usuário, como cliques em botões, preenchimento de formulários e a exibição de conteúdos dinâmicos sem a necessidade de recarregar a página. Além disso, ele permite manipular o DOM (Document Object Model), facilitando a alteração de elementos HTML diretamente a partir do código. Outro benefício do JavaScript é a possibilidade de trabalhar com APIs, que permitem integrar seu site a serviços externos, como sistemas de pagamento, redes sociais ou serviços de geolocalização.


Uma vez confortável com HTML, CSS e JavaScript, é importante aprender a usar o Git. Git é uma ferramenta de controle de versão que ajuda a gerenciar alterações no código ao longo do tempo. Com o uso de Git, é possível colaborar com outros desenvolvedores em projetos compartilhados, rastrear o histórico de modificações no código e reverter alterações quando necessário. Complementando o Git, há o GitHub, que serve como uma plataforma para hospedar seus projetos e colaborar com outros desenvolvedores, além de facilitar a apresentação do seu portfólio para possíveis empregadores ou parceiros.
